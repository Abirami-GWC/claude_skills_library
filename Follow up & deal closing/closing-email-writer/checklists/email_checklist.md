# Closing Email — Self-Check

- [ ] Every bracket in the template filled with a real detail — none left as placeholder text?
- [ ] No invented deadline, discount, or commitment?
- [ ] Recipient name used, or a generic professional greeting if genuinely unknown (never a guessed name)?
- [ ] Length within guideline (3-6 sentences, or recap's slightly longer allowance)?
- [ ] Tone matches `email_guidelines.md` — direct, warm, not pushy?
- [ ] If a `my-writing-style` profile exists, was it applied?
- [ ] Does the email make exactly one clear ask or next step, not several competing ones?
