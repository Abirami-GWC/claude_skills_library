## How proposal-builder should use this

1. Check this table for the item/rate before asking the user for pricing.
2. If listed, use it as the default and confirm quantity/scope only.
3. If not listed, or the deal is custom-negotiated, ask the user directly.
4. Always show discount math explicitly in the pricing table (list price → % off → final total).
