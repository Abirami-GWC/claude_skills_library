# Ad Copy Generator — Checklists

## Headline/Hook Checklist
- [ ] At least 3 genuinely distinct frameworks/angles used (not reworded duplicates)
- [ ] Value proposition clear within the first line
- [ ] Character limits respected for the specified channel
- [ ] Curiosity gaps (if used) pay off honestly in the body copy
- [ ] Specificity used over vague superlatives where possible

## CTA Checklist
- [ ] CTA commitment level matches funnel stage (low-friction for cold, high-commitment for decision/retargeting)
- [ ] CTA leads with a verb
- [ ] No fabricated urgency/scarcity
- [ ] CTA accurately reflects what happens after the click

## Claims & Ethics Checklist
- [ ] No invented statistics, testimonials, certifications, or guarantees
- [ ] No disparagement of specific named real competitors
- [ ] No disproportionate exploitation of fear/insecurity, especially toward vulnerable groups
- [ ] Any urgency/scarcity claims confirmed real before use
- [ ] Required disclaimers (if any) preserved exactly

## Channel Format Checklist
- [ ] Correct channel identified/confirmed before drafting
- [ ] Copy structured to that channel's conventions (not just resized from another channel)
- [ ] Character/field limits respected
- [ ] CTA options match platform constraints (e.g., LinkedIn's fixed CTA button list) where applicable

## Quality / Review Checklist (before returning final deliverable)
- [ ] Multiple genuinely distinct angles delivered (unless a single final version was requested)
- [ ] Each variation includes a brief rationale
- [ ] Framework used for body copy (AIDA/PAS/FAB/etc.) is coherent start to finish
- [ ] Brand voice constraints (if supplied) respected
- [ ] Output is ready to use with minimal further editing
