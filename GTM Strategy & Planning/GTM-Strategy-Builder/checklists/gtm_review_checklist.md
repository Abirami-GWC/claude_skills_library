# GTM Review Checklist

**Purpose of this document:** A final review pass to run before presenting a GTM strategy output to the user — distinct from `strategy_checklist.md`, which checks content completeness while drafting. This checklist checks internal consistency and readiness to finalize.

## Internal Consistency

- [ ] Positioning's target customer matches the ICP's defined segment
- [ ] Messaging pillars are traceable to the positioning's stated differentiator(s)
- [ ] Persona pains/gains referenced in messaging match the personas actually defined
- [ ] Channel/motion choices match where the defined persona(s) actually research and buy
- [ ] No contradictory claims exist across positioning, messaging, and channel sections

## Industry-Agnostic and Reusability Check

- [ ] No real, specific company or brand name is used as if it were the user's own company
- [ ] No real company is cited as a factual example/case study (only generic/hypothetical illustrations)
- [ ] Language is free of assumptions that only apply to one specific industry unless the user's context is genuinely that industry

## Scope Discipline

- [ ] No dollar-specific budget allocation is included (that belongs to the Budget Allocator skill)
- [ ] No detailed execution timeline/sprint plan is included (that belongs to the Quarterly Roadmap Planner skill)
- [ ] No board-facing financial narrative or slide structure is included (that belongs to the Board Deck Narrative skill)
- [ ] Any adjacent request beyond this skill's scope is explicitly flagged with the correct companion skill named

## Evidence and Assumption Transparency

- [ ] Every major claim is labeled as either user-provided fact, reasonable inference, or explicit assumption
- [ ] Any unvalidated messaging/positioning claim is explicitly flagged as such, not presented as confirmed
- [ ] Recommended validation steps (e.g., win/loss interviews, customer validation) are included wherever hypothesis-only content exists

## Actionability

- [ ] Every strategic recommendation is specific enough for a marketer or seller to act on within a week
- [ ] The number of recommended channels is focused (2-4), not an unprioritized exhaustive list
- [ ] The number of messaging pillars is focused (3-4), not sprawling
- [ ] Quarterly goals are measurable, not vague aspirations

## Final Presentation Readiness

- [ ] Output uses the correct template(s) from `templates/` for the artifact(s) requested
- [ ] All placeholders have been replaced with drafted content (no remaining bracketed placeholders)
- [ ] A short summary of key strategic choices and flagged assumptions is included alongside the full artifact
- [ ] Output has passed `checklists/strategy_checklist.md` and `checklists/quality_checklist.md`
