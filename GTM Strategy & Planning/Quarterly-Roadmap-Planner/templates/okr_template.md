# OKR Template

> **Purpose:** Fill-in-the-blank template for drafting objectives and key results cascaded from the quarterly roadmap's themes. See `references/okr_framework.md` for methodology.

## Instructions

- One block per team/function per objective. Limit each team to 2-4 objectives per quarter.
- Each objective should have 2-4 key results. Every key result must be numeric and gradable.
- Complete the "Grading" section at quarter close-out, not at drafting time.
