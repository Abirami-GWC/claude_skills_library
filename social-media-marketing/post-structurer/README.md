# Post Structurer

Turns one topic, idea, or rough draft into a complete, ready-to-publish social media post — hook, body, and CTA — correctly shaped for the target platform.

## What it does
Takes a single idea (or a rambling draft) and rewrites it into a tight post using one of four proven structural frameworks, formatted according to the target platform's length and line-break norms.

## When it triggers
- "Turn this into a LinkedIn/Instagram/X post"
- "Structure this post for me"
- "Write out this idea as a full post"
- A rough draft that needs tightening into hook-body-CTA shape

## Folder contents
- `SKILL.md` — the skill definition and workflow Claude follows
- `knowledge/structure-frameworks.md` — the four core structural frameworks and when to use each
- `mappings/platform-format-mapping.md` — length, line-break style, and CTA placement per platform
- `templates/post-structure-template.md` — internal drafting scaffold
- `examples/example-run.md` — a full worked example (rough input → finished post)
