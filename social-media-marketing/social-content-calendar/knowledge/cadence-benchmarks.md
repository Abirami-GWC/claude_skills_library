# Posting Cadence Benchmarks

Use these as sanity-check ranges, not hard rules — the user's actual capacity matters more than a "best practice" number.

| Platform | Realistic minimum (small team/solo) | Sustainable sweet spot | Aggressive (needs dedicated resource) |
|---|---|---|---|
| LinkedIn (company or personal) | 2x/week | 3-5x/week | Daily |
| Instagram (feed/carousel) | 2x/week | 3-4x/week | Daily + multiple Stories |
| Instagram Reels | 1x/week | 2-3x/week | Daily |
| X / Twitter | 3x/week | 1x/day | Multiple/day |
| TikTok | 2x/week | 3-5x/week | Daily |
| Facebook | 2x/week | 3x/week | Daily |

## Guidance
- If the user proposes a cadence far above "sustainable sweet spot" without a dedicated content person, flag it — a calendar that gets abandoned in week 2 is worse than a smaller one that actually runs.
- Consistency beats frequency: 3 posts/week sustained for months outperforms 7 posts/week for two weeks then silence.
- New content pillars or new platforms should start at the lower end of the range and scale up once a rhythm is established.
