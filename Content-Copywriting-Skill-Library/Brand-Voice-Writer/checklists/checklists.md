# Brand Voice Writer — Checklists

## Brand Voice Checklist (use when drafting or rewriting)
- [ ] Voice fingerprint identified (personality, spectrum placements, do/don't examples)
- [ ] Source of voice signal noted (examples vs. adjectives vs. style guide) and conflicts flagged
- [ ] Sentence length/rhythm matches established pattern
- [ ] Point of view (I/we/you/third person) consistent with voice
- [ ] Contraction usage matches voice (present if casual, absent if formal, per source)
- [ ] Vocabulary matches established jargon tolerance and figurative language use
- [ ] No generic/AI-neutral phrasing has crept in (e.g., "in today's fast-paced world")

## Tone Checklist (use when situational context matters)
- [ ] Situation type identified (celebration / everyday / complaint / error / legal / sales / sensitive)
- [ ] Energy/formality level matches the tone matrix for that situation
- [ ] Humor dialed appropriately (down or removed in negative/serious contexts)
- [ ] Empathy and clarity prioritized in negative-state content
- [ ] Some brand personality marker retained even at the most restrained end
- [ ] No jokes at the reader's expense in complaint/error contexts

## Messaging Checklist
- [ ] Relevant messaging pillar(s) identified for this piece's purpose
- [ ] All claims are backed by proof points present in source material — nothing invented
- [ ] Audience framing (how readers are referred to) consistent with brand preference
- [ ] Piece doesn't try to cram in every pillar at once

## Editorial / Quality Checklist
- [ ] Grammar, tense, and point-of-view consistent throughout
- [ ] Terminology consistent (same term for same concept throughout)
- [ ] Formatting consistent (heading case, list style)
- [ ] Active voice used except where passive is a deliberate stylistic choice
- [ ] No buried verbs or unnecessarily stacked prepositional phrases
- [ ] Sentence length varied, not monotonous

## Review / QA Checklist (before returning any final deliverable)
- [ ] Meaning and factual claims match the source exactly
- [ ] No banned words/phrases present
- [ ] Required terminology present and used correctly
- [ ] Rationale for changes is specific, not generic
- [ ] Output would pass as written by someone deeply familiar with the brand
- [ ] Any inferred (vs. confirmed) elements are flagged to the user
