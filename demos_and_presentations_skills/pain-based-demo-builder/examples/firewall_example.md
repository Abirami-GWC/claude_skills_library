# Example: Firewall / Network Security

**Input**
Customer: Manufacturing company
Pain Points:
- Frequent malware incidents on the shop floor network
- No segmentation between IT and OT networks
- IT team overwhelmed managing firewall rules manually

**Output**
Demo Agenda
1. Network segmentation (IT/OT separation)
2. Automated threat detection & blocking
3. Simplified, centralized firewall rule management
4. Customer success story (manufacturing sector)
5. Q&A

**Why this order:** Segmentation is foundational — it limits blast radius before
showing detection capabilities. Automated detection addresses the stated malware
pain directly. Rule management closes on reducing IT burden, which is a real but
secondary pain compared to active incidents.
