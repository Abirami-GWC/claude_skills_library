# Example: Endpoint Security

**Input**
Customer: Healthcare provider
Pain Points:
- Ransomware concerns across clinic laptops
- No visibility into endpoint activity
- Slow to detect lateral movement

**Output**
Demo Agenda
1. Endpoint detection & response (EDR) dashboard
2. Behavioral anomaly detection (lateral movement)
3. Automated ransomware containment
4. Customer success story (healthcare sector)
5. Q&A

**Why this order:** Visibility is shown first since it's foundational to everything
else the customer needs to trust. Anomaly detection addresses the lateral-movement
pain directly, and containment closes on the customer's top-of-mind fear
(ransomware) right before the proof point.
