# Cloud Playbook

## Pain: Migration risk / unfamiliar with cloud

**Demo Flow**
1. Show guided migration assessment
2. Show phased migration plan
3. Show hybrid environment visibility (cloud + on-prem)

**Business Value**
Lower migration risk; predictable timeline; no visibility gap during transition.

## Pain: Unpredictable cloud costs

**Demo Flow**
1. Show cost visibility dashboard
2. Show cost optimization recommendations
3. Show budget alerts / forecasting

**Business Value**
Predictable spend; identified savings opportunities.

## Pain: Scaling is manual and slow

**Demo Flow**
1. Show auto-scaling configuration
2. Show load-based scaling triggers
3. Show performance monitoring during scale events

**Business Value**
Faster response to demand spikes; reduced manual ops work.

## Sequencing note
If migration risk is a stated pain, it goes first (it's blocking everything
else). Otherwise, lead with whichever of cost/scaling was mentioned with more
urgency or frequency.
