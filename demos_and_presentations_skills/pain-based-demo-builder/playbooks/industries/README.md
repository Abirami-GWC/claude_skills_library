# Industry Playbooks

Category playbooks (in the parent `playbooks/` folder) answer "given this pain,
what do we demo?" Industry playbooks answer "given this industry, how should we
weight and frame those pains?"

Add one file per industry you sell into regularly. Each should cover:
- The typical top pains/priorities for that industry, in rough order
- Framing/language guidance
- Which case studies to prefer

Currently available: `manufacturing_playbook.md`. Add others (healthcare, retail,
financial services, etc.) as needed — follow the same structure.
