# Recommendation (Per-Deal Explanation) Template

Use this pattern for the "Why this rank" cell/line for each deal. The goal: a rep
should be able to defend the order to their manager just from this sentence.

## Pattern

`[Rank word/number] — [1-2 driving factors], [1 qualifying/contrasting factor if relevant].`

## Good examples

- "Ranked #2 — large deal and closing this week, despite being early-stage."
- "Ranked #8 — solid probability, but no contact in 3+ weeks is dragging it down."
- "Ranked #14 — mid-size and mid-stage, no red flags but nothing pulling it up either."
- "Tied with [Deal] — similar value and stage, both worth attention this week."

## Avoid

- Generic filler that doesn't reference the actual data: "This deal is important and
  should be prioritized."
- Repeating every sub-score: don't list value/probability/urgency/engagement/stage
  for every single deal — name only what actually moved the needle.
- Vague hedging: "might need some attention" — say what specifically, based on which
  field.

## When a deal implies a clear next step

It's fine to hint at it ("hasn't been touched in 3 weeks") but don't turn this into a
full action plan — detailed next-step scheduling belongs to
`follow-up-cadence-builder`, not this skill.
