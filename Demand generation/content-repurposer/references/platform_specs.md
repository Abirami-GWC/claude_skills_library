# Platform Specs — Length Limits and Conventions

| Platform / Format | Length limit | Structural convention |
|---|---|---|
| X / Twitter (single post) | 280 characters | Hook must work standalone; hashtags sparingly (0–2) |
| X / Twitter (thread) | 280 chars/post | Post 1 must earn the expand-click; number posts if long (1/, 2/...) is optional, not required |
| LinkedIn (single post) | ~3,000 characters, but first ~200 characters shown before "see more" | First 1–2 lines must hook before the fold |
| LinkedIn (carousel/thread) | Per-slide text kept short (~1 idea per slide) | Slide 1 = hook, last slide = CTA |
| Instagram caption | 2,200 characters max, but effective length ~125 chars before truncation | Hook line first; hashtags at end (not inline) |
| Email subject line | 40–60 characters for full visibility across clients | Curiosity or benefit-driven, avoid spammy-trigger words |
| Email teaser/snippet body | 60–90 words typical for a newsletter blurb | One idea, one CTA |
| Short video / reel script | 15–30 seconds spoken (~40–75 words) | 3-second hook, one point, one CTA |
| Infographic outline | N/A (outline, not prose) | 1 anchor stat + 3–5 supporting points |

Always verify actual character/word count before delivering — count, don't estimate.

## Notes on volatility

Platform limits change periodically (character limits, feature support). If
the user's request depends on a very recent or platform-specific limit change
not reflected here, verify current limits via a web search rather than
relying solely on this reference — flag to the user if you're not certain the
listed limit is still current.
