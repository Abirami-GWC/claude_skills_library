# Validation Rules — Lead Magnet Builder

Run before delivering any lead magnet and its landing page copy.

## Value/scope checks

- [ ] The offer's title/headline states one clear, specific promise.
- [ ] The content actually delivers what the title promises — no scope claimed
      that isn't covered.
- [ ] No padding — every section/item serves the stated promise; nothing
      included just to add length.

## Format-specific checks

- [ ] **Checklist**: every item is a concrete, checkable action (starts with
      a verb, has a clear completion state) — not a restated goal.
- [ ] **Guide**: sections are scoped to the promise, not a generic overview of
      the whole topic area.
- [ ] **Calculator**: the formula/logic was explicitly confirmed with the
      user before building — never fabricated. The built tool was tested
      against at least one sample input and produces the expected output.
- [ ] **Template/Worksheet**: fields reflect the specific methodology being
      taught, not a generic blank document.

## Landing page copy checks

- [ ] Copy sells the specific outcome (from Step 1's value statement), not
      just the format ("checklist," "guide").
- [ ] No unverifiable or exaggerated claims ("guaranteed," "the only way to...").
- [ ] Form fields requested match what's actually needed for follow-up — flag
      if the user's form asks for more than the offer justifies.

## Final gate

If the calculator's formula wasn't explicitly confirmed with the user, do not
build or deliver it — go back and confirm first. This is the single
highest-risk failure mode in this skill, since a wrong or fabricated formula
actively misleads whoever uses the tool.
