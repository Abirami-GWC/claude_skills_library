# Goal → CTA Type Mapping

| Stated goal | Primary CTA type | Best-fit platforms | Notes |
|---|---|---|---|
| Comments / discussion | Comment-driving | LinkedIn, X, Facebook | Best on opinion or results posts where a specific question fits naturally |
| Reach / virality | Share-driving | LinkedIn, Facebook, X threads | Works best when the post has clear "send to a friend/colleague" value |
| Saves / being a reference | Save-driving | Instagram, LinkedIn (carousels) | Best on educational/framework/checklist content |
| Growing audience | Follow-driving | Instagram, TikTok, X | Only works if there's a real recurring value proposition, not just "more content" |
| Traffic / clicks | Click-driving | LinkedIn (comment-link), Instagram (bio-link), X (inline link) | Respect platform link norms — see post-structurer's platform-format-mapping for link placement rules |

## Decision rule
If the user hasn't stated a goal, infer it from the post content:
- Story/personal content → default to comment-driving (builds discussion)
- Framework/how-to content → default to save-driving
- Results/case-study content → default to share-driving
- Recurring content pillar → follow-driving is worth offering as a secondary option
