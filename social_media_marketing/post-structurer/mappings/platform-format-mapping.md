# Platform → Format Mapping

| Platform | Target length | Line-break style | CTA placement |
|---|---|---|---|
| LinkedIn | 150-300 words | Short paragraphs (1-3 lines), blank line between each | End of post; links go in first comment, not inline |
| Instagram (caption) | 50-150 words for feed post, up to 300 for educational carousel captions | Short lines, can use light emoji as visual breaks if it matches brand voice | End of caption; "link in bio" if needed |
| X / Twitter (single) | Under 280 characters, one idea | N/A — single block | Often implicit (reply/quote) rather than a hard ask |
| X / Twitter (thread) | First tweet ≤ 250 characters; each subsequent tweet one idea | New tweet per beat | Final tweet: clear ask (follow, reply, link) |
| TikTok (script) | 15-45 seconds spoken (roughly 40-110 words) | Written as spoken lines with on-screen text cues in brackets | Verbal CTA in final 2-3 seconds |
| Facebook | Similar to LinkedIn but slightly more casual, can run longer for community posts | Short paragraphs | End of post, can be softer/more conversational |

## Notes
- Never inline a raw URL in the main body of a LinkedIn or Facebook post if the goal is reach — note it belongs in the first comment instead, and mention this to the user.
- For carousels, structure output as "Slide 1: ... Slide 2: ..." rather than one continuous caption.
- For TikTok, always include bracketed on-screen text cues alongside spoken lines, e.g. `[on-screen: "40% drop in signups"]`.
