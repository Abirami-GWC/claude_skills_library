# Lead Magnet Types

## Guide / eBook
- **Best for**: topics requiring explanation and context, not just a list of steps
- **Length**: as long as needed to deliver the promise, no longer — a focused
  6-page guide beats a padded 20-page one
- **Risk**: easiest format to pad with filler; keep every section tied to the
  stated promise

## Checklist
- **Best for**: process-driven topics where the value is completeness/order
  ("did I miss anything")
- **Length**: typically 1 page; each item must be a concrete, checkable action
- **Risk**: vague items that restate a goal instead of an action (see SKILL.md
  Best Practices)

## Template / Worksheet
- **Best for**: topics where the reader needs a fill-in-the-blank starting
  point (a plan, a framework, a tracking sheet)
- **Length**: structured document with clear fields/sections to complete
- **Risk**: too generic to be useful — should reflect the specific
  methodology/framework being taught, not a blank generic document

## Interactive Calculator
- **Best for**: topics where the value is a personalized number (ROI,
  savings, sizing/capacity estimate, pricing estimate)
- **Length**: N/A — built as a working tool, not a document
- **Risk**: highest-risk format if the underlying formula isn't real/sourced
  — never fabricate the calculation logic; always confirm with the user

## Quiz / Assessment
- **Best for**: topics where the value is self-diagnosis (what stage/type/
  maturity level is the reader at)
- **Length**: 5–10 questions with a scored/segmented result
- **Risk**: results must be genuinely differentiated per answer pattern, not
  a single generic result regardless of input

## Selecting a format

| If the value is... | Use |
|---|---|
| "Here's everything you need to know about X" | Guide |
| "Make sure you don't miss these steps" | Checklist |
| "Here's a starting structure to fill in" | Template/Worksheet |
| "Here's your specific number" | Calculator |
| "Here's where you stand / what type you are" | Quiz |

If the user hasn't specified a format, propose the one that best matches
their stated value proposition and confirm before drafting.
