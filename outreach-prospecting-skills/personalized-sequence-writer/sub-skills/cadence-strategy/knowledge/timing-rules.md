# Timing Rules

Base touch counts and spacing by deal-size band (before persona/warmth adjustments).

| Band | Base touches | Base spacing (days between touches) |
|---|---|---|
| < $5k | 3 | 2, 3 |
| $5k-$25k | 5 | 3, 4, 5, 6 |
| $25k-$100k | 6 | 3, 4, 5, 6, 7 |
| > $100k | 7 | 3, 4, 5, 7, 7, 8 |

## Adjustments
- VP/C-level persona: subtract 1-2 touches, keep spacing the same or slightly longer.
- Warm signal present: add 1 touch, can start with a slightly shorter first gap (2 days).
- Fully cold + low context available: do not add touches — more touches without
  substance produces more noise, not more replies.

## Worked example
Deal size $30k, persona = Director, cold outreach:
- Base: 6 touches, spacing 3/4/5/6/7
- No adjustment for persona (Director, not VP/C-level)
- No warm-signal adjustment (cold)
- Result: Day 0, 3, 7, 12, 18, 25 — 6 touches over 25 days.
