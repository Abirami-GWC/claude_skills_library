# Example: SASE / Cloud-Delivered Networking

**Input**
Customer: Multi-site retail chain
Pain Points:
- Inconsistent security policy across 200+ store locations
- Slow, complex site rollouts for new stores
- Cloud app performance issues at branch locations

**Output**
Demo Agenda
1. Centralized policy management across all sites
2. Templated, rapid new-site deployment
3. Optimized cloud application performance at the edge
4. Customer success story (multi-site retail)
5. Q&A

**Why this order:** Policy consistency is foundational (it's a security and
compliance risk across all sites), rollout speed is the next operational pain,
and performance optimization closes as the "make it faster" layer once
consistency and speed are addressed.
